**MscaleFNO** extends the Fourier Neural Operator framework for learning *high-frequency wave scattering* problems.  
It introduces hierarchical frequency decomposition, adaptive spectral filters, and multi-scale architectures for oscillatory PDEs.

This repository includes PyTorch implementations and MATLAB utilities for dataset generation.
